    Get started
    New a project, select Empty Activity, enter package name, select kotlin, enter the game name as Time Fighter, and try to run the app to show “hello world” on the screen.

    1. Create new project : enter the name – Time Fighter, your package name

    2. As default

    3. Empty Activity

    4. Press “run” arrow in the right-top corner

    5. Wait and get the “hello world”

    Add UI
    2.1 Add TextView: your score
    2.2 Add TextView : time left
    2.3 Positioning Items (your score & time left)
    2.4 Adding a Constraint: tap me

    1. Open the activity_main.xml

    2. Drag the Textview to the left corner and the right corner

    3. Delete the “hello world” and drag the button to the center

    4. Positioning each item to correctly shown

    5. Fill in the elements for the three items
      Top-left: your score (Textview) enter id & text

    Top-right: left time (Textview) enter id & text

    Center : TapMe (button) enter id & text

    Add string name in the string.xml
    Your Score: %1$d
    Time Left: %1$d

    3.5 Starting and Ending the Game

    1. Add string name in the strings.xml
      Times up! Your score was: %1$d

    2. Add gameStarted definition

    1. Add startGame in the incrementScore function
      if(!gameStarted){
      startGame()
      }

    Define startGame(press alt+enter, create function)
    private fun startGame() {
    countDownTimer.start()
    gameStarted = true
    }

    1. Add endGame in the resetGame function -> onFinish()
      endGame()

    And endGame function(import the Toast by pressing alt+enter)
    private fun endGame() {
    Toast.makeText(this, getString(R.string.gameOverMessage, score), Toast.LENGTH_LONG).show()
    resetGame()
